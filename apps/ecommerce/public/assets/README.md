# Assets Ecommerce

Tempat penyimpanan aset visual untuk halaman ecommerce.

## Struktur Folder

- `hero/` : gambar hero atau header utama.
- `product/` : gambar produk (thumbnail/card/detail).
- `kategori/` : gambar kategori favorit.
- `inspiration/` : gambar section inspiration/lookbook.

## Rekomendasi Ukuran

- Hero: minimal `1440x600` (WebP/JPG).
- Product: `800x800` (rasio 1:1).
- Kategori: `600x600` (rasio 1:1).
- Inspiration: `1200x800` (bebas, konsisten per template).

## Contoh Penamaan

- `hero/hero-main.webp`
- `product/denim/denim1.jpg`
- `product/dress/dress1.jpg`
- `product/bag/bag1.jpg`
- `product/sneakers/sneakers1.jpg`
- `kategori/kategori-wanita.webp`
- `inspiration/inspiration-01.webp`

## Cara Panggil di Frontend

Contoh path dari komponen:

- `/assets/hero/hero-main.webp`
- `/assets/product/denim/denim1.jpg`
- `/assets/kategori/kategori-wanita.webp`
- `/assets/inspiration/inspiration-01.webp`
